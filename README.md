# Forgotten
This is a fictional case-file created for my biology class. It has some research notes on a newly discovered but now forgotten species called the Arctic Frost Owl. The site is made with vanilla HTML, CSS and some Javascript. It's hosted on netlify.

## Local development
Just clone the repo and open the index.html file in your browser. I use vscode with the live server extension for development.
